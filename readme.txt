== Description ==
用于快速启动项目的页面类型。
方便添加产品、或生产相关的内容，不影响帖子和页面的权限管理，适合用于制作生产相关的营销落地页。

* 分配的权限独立：
例如，你可以分配相同的工作组给不同的人，可以是product类别，一个管理文章，一个管理产品上传。

* 自定义设置：
根据你的需要修改相关设置，定制你自己的内容组。

代码在插件中，使用最精简的代码制作，插件不会因为Wordpress版本更新受影响。

1.0 只考虑有产品添加时的情况；
2.0 因为面向的B端情景较多，因此做了面向制造商的优化。

适合场景：
1.快速启动站点；
2.分配给下属员工权限；
3.制作落地页方便字段调用和特别类型的内容管理。（装个权限管理插件分配就行了。）

== Installation ==
直接上传压缩包安装，后缀名必须是ZIP格式。
